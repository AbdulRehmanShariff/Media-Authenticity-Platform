# Media Authenticity Platform (Deepfake Detection System)

A unified full-stack application designed to detect digital manipulation in images, videos, and audio, as well as analyze text for misinformation using AI.

![Project Status](https://img.shields.io/badge/Status-Complete-success)
![Tech Stack](https://img.shields.io/badge/Stack-React_|_Flask_|_TensorFlow-blue)

## üöÄ Features

* **Image Detector:** Analyzes pixel-level artifacts using an **Xception-based CNN** (92% Accuracy).
* **Video Detector:** Frame-by-frame analysis using **EfficientNetV2** and OpenCV.
* **Audio Detector:** Detects synthetic voices using a hybrid **Conv1D + LSTM** network on MFCC spectrograms.
* **Misinformation Checker:** Forensic text analysis powered by **Google Gemini AI**.
* **AI Assistant:** An integrated chatbot for technical project walkthroughs.
